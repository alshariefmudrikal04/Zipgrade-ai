ZipGrade AI

ZipGrade AI is a simple web application that extracts and parses multiple-choice questions (MCQs) from PDF files and optionally generates answers using AI. Inspired by ZipGrade, it helps convert PDF quizzes and reviewers into structured, scannable answer sheets.
